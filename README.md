# 🏰 Age of Empires II Collection

A curated bundle of classic **Age of Empires II** content—bringing together the original experience, expansions, and a legendary community mod.

---

## 📦 Includes

### 🎮 Base Game

* **Age of Empires II: The Age of Kings**
  The original real-time strategy classic that started it all.

### ⚔️ Official Expansion

* **Age of Empires II: The Conquerors**
  Expands the base game with new civilizations, campaigns, and mechanics.

### 🛠️ Community Expansion

* **Age of Empires II: Forgotten Empires** *(Unofficial Mod)*
  A fan-made expansion that adds fresh content, civilizations, and gameplay enhancements.

🔗 Install guide:
https://www.forgottenempires.net/install

---

## 📁 Side Content

* **Age of Empires II: HD Edition (2013 / Retired)**
  Provided as a `.rar` archive for legacy or compatibility purposes.

---

## ⚙️ Notes

* **Forgotten Empires** is preinstalled with the base game and **The Conquerors** expansion.
* The "fe_update" file is the installation process for the **Forgotten Empires* found at https://www.forgottenempires.net/install
* The HD Edition is included for archival or optional use.
* Some components may require compatibility settings depending on your system.
