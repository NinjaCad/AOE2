# 🏰 Age of Empires II Collection

A curated bundle of classic **Age of Empires II** content—bringing together the original experience, its iconic expansion, and a legendary community mod.

---

## 📦 Includes

### 🎮 Base Game

* **Age of Empires II: The Age of Kings**
  The original real-time strategy classic that started it all. Build empires, command armies, and shape history.

### ⚔️ Official Expansion

* **Age of Empires II: The Conquerors**
  Expands the base game with new civilizations, campaigns, units, and deeper gameplay mechanics.

### 🛠️ Community Expansion

* **Age of Empires II: Forgotten Empires** *(Unofficial Mod)*
  A fan-made expansion packed with new civilizations, campaigns, balance changes, and quality-of-life improvements.

🔗 Official install guide:
https://www.forgottenempires.net/install

---

## 📁 Side Content

* **Age of Empires II: HD Edition (2013 / Retired)**
  Included as a `.rar` archive for legacy support, archival purposes, or optional use.

## ⚙️ Setup Instructions

1. 📂 Move the folder named **"Microsoft Games"** into:

   ```
   C:\Program Files (x86)\
   ```

2. 📁 Open the following directory:

   ```
   C:\Program Files (x86)\Microsoft Games\Age of Empires II\age2_x1
   ```

3. ▶️ Launch the game by running:

   ```
   age2_x2.exe
   ```

---

## ⚙️ Additional Notes

* ✅ **Forgotten Empires is already preinstalled** with the base game and **The Conquerors** expansion.
